## งานที่ทำใน session นี้

### 1. Fix `timeAgo` NaN bug (`OverviewPage` + `Topbar`)

**Root cause:** .NET คืน date format `"M/D/YYYY h:mm:ss AM"` — `new Date(ts + 'Z')` ทำให้ parse ผิด

**Fix:**
- ลบ `+ 'Z'` ออก — ใช้ `new Date(ts)` ตรง ๆ แล้วเช็ค `isNaN(d.getTime())`
- แก้ใน `OverviewPage.tsx` (timeAgo ภาษาไทย) และ `Topbar.tsx` (timeAgo ภาษาอังกฤษ)

### 2. Fix Breadcrumb — แสดงชื่อจริงจาก API

**Root cause:** `useBreadcrumbs()` ใช้ hardcoded dicts (`BUILDING_LABEL`, `FLOOR_LABEL` ฯลฯ) ที่ออกแบบสำหรับ mock ID รูปแบบ `a-f1`, `hq`

**Fix:** เขียน `useBreadcrumbs()` ใหม่ให้ใช้ `useQuery` 3 ตัวคู่ขนาน:
- `getFloorById(id)` — โหลดชื่อ floor + `Building_ID`
- `getBuildingById(buildingId)` — โหลดชื่อ building + `Site_ID`
- `getSites()` — map `Site_ID` → ชื่อ site

**ผล:** breadcrumb แสดง "Home / สำนักงานใหญ่ / อาคาร A / ชั้น 1" ✅

### 3. Fix BuildingMapPage blank page

**Root cause:** `@react-leaflet/core` ถูก install เป็น separate dependency — react-leaflet v4 bundle มันมาให้อยู่แล้ว การ install ซ้ำทำให้เกิด duplicate React Context → `render2 is not a function` crash

**Fix:**
- ลบ `@react-leaflet/core` ออกจาก `package.json`
- `npm install --legacy-peer-deps`
- Restart Vite dev server (ล้าง dep cache)
